# NyraAI 🧠 
AI Contract Risk Intelligence


**Privacy-First • Explainable • Modular AI Contract Analyzer**

AI Contract Risk Intelligence is an **AI-powered LegalTech SaaS platform** that automatically analyzes contracts and detects risky clauses, liabilities, and compliance issues.

Upload a contract → Receive **risk insights, clause explanations, and improvement suggestions in seconds.**

# 📌 Key Features

### 📄 Contract Upload

Upload contracts in **PDF / DOC / DOCX** format.

### 🧠 AI Clause Analysis

AI identifies:

* Liability risks
* Missing protections
* Unfair clauses
* Compliance gaps

### 📊 Risk Intelligence Dashboard

Visual insights:

* Contract risk score
* Clause risk levels
* Red-flag alerts
* Analytics charts

### 🔐 Privacy-First AI

Supports:

* OpenAI API
* Local LLM mode

Contracts can be analyzed **without sending data externally**.

### 🛡 Enterprise Security

Includes:

* OAuth Authentication
* Role Based Access Control
* Supabase Row Level Security
* Prompt Injection Defense
* Rate Limiting
* Cloudflare WAF
